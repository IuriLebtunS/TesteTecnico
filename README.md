Este projeto é um sistema web para gerenciar lançamentos financeiros. Ele permite adicionar, editar e listar lançamentos, cancelar ou marcar pagamentos, calcular automaticamente valores com taxas ou descontos, visualizar o saldo total e exportar os dados por mês em Excel.  

O front-end foi desenvolvido em ASP.NET Web Forms, o back-end em C# usando uma Class Library, e o banco de dados é SQL Server acessado diretamente via ADO.NET.  

Para rodar o projeto, basta criar o banco de dados usando o script incluído, configurar a `connection string` no `Web.config`, abrir o projeto no Visual Studio e executá-lo.

Desenvolvido como desafio técnico para uma vaga de **nível pleno**, com foco em clareza de código e separação de camadas (WebApp / Business / Data).
