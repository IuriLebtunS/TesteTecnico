# Sistema de Lançamentos Financeiros

Aplicação web para registrar e gerenciar lançamentos financeiros de forma simples.

## Funcionalidades
- Adicionar, editar e listar lançamentos
- Cancelar ou marcar um lançamento como pago
- Cálculo automático de valor com taxa ou desconto
- Visualização do saldo total
- Exportação de lançamentos por mês (PDF, Excel ou CSV)

## Tecnologias
- Front-end: ASP.NET Web Forms
- Back-end: C# (Class Library)
- Banco de dados: SQL Server via ADO.NET

## Como rodar
1. Criar o banco de dados e rodar o script SQL incluído
2. Configurar a `connection string` no `Web.config`
3. Abrir o projeto no Visual Studio e executar
